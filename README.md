# QuteMol

**QuteMol** is an open source (GPL), interactive, high quality molecular visualization system. It exploits current GPU capabilities through OpenGL shaders to offer an array of innovative visual effects aimed at improving clarity and understanding of the 3D shape and structure of large molecules or complex proteins.

## Key Features

- **Real-Time Ambient Occlusion**: Enhances depth perception and structural detail.
- **Depth Aware Silhouette Enhancement**: Improves clarity of complex structures.
- **Visualization Modes**: Supports Ball and Sticks, Space-Fill, and Liquorice modes.
- **High Resolution Snapshots**: Create publication-quality antialiased renderings.
- **Animated GIFs**: Automatically generate rotating molecule animations for the web.
- **Large Molecule Support**: Real-time rendering of molecules with over 100k atoms.
- **Standard Input**: Supports PDB (Protein Data Bank) format.
- **Cross-Platform**: Available for Windows and Mac OS X.

## Hardware Requirements

QuteMol requires graphics hardware that supports:
- OpenGL Shading Language (GLSL)
- Fragment Shaders
- Framebuffer Objects (FBO)

It is recommended to use the latest graphics drivers from your manufacturer.
